## 📄 Overview

<a href="#astuteGraphics">Astute Graphics</a> serves professional Adobe Illustrator users — logo designers who need precision path and corner control, illustrators who use variable-width strokes for expressive line quality, type designers working with precise geometric construction, and any Illustrator user whose work has run up against the limitations of Illustrator's native toolset.

The <a href="#astuteGraphics">logo designer use case</a> benefits most from VectorScribe and MirrorMe — Dynamic Corners for the clean, adjustable rounded corners that mark professional logo geometry, and MirrorMe for symmetric logo construction where both sides must be identical and any refinement should apply symmetrically. The combination makes logo geometry work that Illustrator does awkwardly into work that Illustrator does beautifully. The <a href="#astuteGraphics">illustrator and lettering artist use case</a> uses WidthScribe for the organic width variation that distinguishes hand-lettered work from mechanically uniform strokes — painting width variations across letterforms with the interactive immediacy that Illustrator's width profile tools alone don't provide.

The <a href="#astuteGraphics">editorial and technical illustrator use case</a> uses Phantasm for color adjustment flexibility and Stipplism for the traditional etching and stipple illustration styles that editorial contexts often require — applying printmaking aesthetics to vector artwork without destructive effects. <a href="#astuteGraphics">InkScribe plugin</a> provides a complete replacement drawing tool for Illustrator's Pen tool — smoother curve preview during drawing, easier anchor point manipulation, and drawing behavior that many designers find more intuitive than Illustrator's standard Pen.

<a href="#astuteGraphics">DirectPrefs plugin</a> provides quick access to Illustrator preferences that are buried in menus — toggling grid, snapping, rulers, and other preference states from keyboard shortcuts rather than navigating preference dialogs. <a href="#astuteGraphics">Plugin compatibility with Creative Cloud</a> updates are maintained by Astute Graphics with each major Illustrator release — the plugins remain compatible as Illustrator updates through Creative Cloud, with Astute Graphics providing compatibility updates promptly after major Illustrator versions release.

<a href="#astuteGraphics">Trial availability</a> for all Astute Graphics plugins allows evaluation before purchase — testing each plugin in the actual Illustrator workflow before committing to subscription or individual plugin purchase. <a href="#astuteGraphics">Active tutorial and learning resources</a> from Astute Graphics — video tutorials for every plugin covering workflows from basic to advanced make learning each tool's specific capabilities accessible.

## 🤔 FAQ

<details>
<summary>What is ColliderScribe used for?</summary>

ColliderScribe provides precise spatial relationship tools — exact gap control between objects, collision detection for tight spatial arrangements, and snapping behavior beyond Illustrator's standard tools.

</details>

<details>
<summary>Does SubScribe add construction geometry tools?</summary>

Yes. SubScribe adds tangent line drawing, angle bisection, and other geometric construction tools for technical illustration and precise ornamental design.

</details>
